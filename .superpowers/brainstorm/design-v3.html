设计说明
1
顶部统计卡：4 个指标卡（总技能数、激活工具数、同步状态、最近更新），扫一眼知道整体健康状态。
2
筛选栏：搜索 + 快捷过滤（All / Git / Local）+ 排序，操作集中，不跳页。
3
技能卡片展开同步控制：点击任意技能卡片 → 展开同步面板，显示所有工具 toggle + Sync / Update / Remove 操作。点击 react-best-practices 的标题行可体验收起/展开。
4
左侧图标导航：常驻 My Skills / Explore / Settings，后续可加更多模块。hover 显示 tooltip。
